Interleaved review

Re-reading the lessons feels like learning but mostly builds fluency, the illusion of mastery. Retrieving from memory, cold, with the principles shuffled, is what builds storage strength. That's what this page is for.

How to use this

Questions are drawn from all seven lessons and shuffled every run, on purpose. Interleaving the principles rather than reviewing one at a time is harder than blocked practice, and that difficulty is exactly what makes it stick.1 Many questions pit the right answer against a near-miss: the shallow version that sounds right. Answer from memory before reading the feedback. Then hit Shuffle & retry, and run it again in a few days — that spacing is the second half of durable memory.

Go deeper. Missed the same principle twice? Flag it — it's worth re-explaining a different way, or running a live board state through the chain to drill just that step. When you can clear this cold two sessions apart, you can run value → cost → space → concentration → timing on any position without notes.
