# 5x5 Cube Resources

## Knowledge

- [J Perm — How To Solve the 5x5 Rubik's Cube](https://jperm.net/5x5)
  The primary source for this course. Clear reduction walkthrough (centers → edges → 3x3), video plus written notation, and the exact parity algorithm (`Rw U2 x Rw U2 Rw U2 3Rw' U2 Lw U2 Rw' U2 Rw U2 Rw' U2 Rw'`, verified 2026-07-09). Use for: every stage of the beginner solve.
- [Speedsolving Wiki — Reduction Method](https://www.speedsolving.com/wiki/index.php/Reduction_Method)
  The canonical description of reduction: dominant method for 5x5 and larger, three stages, three algorithms (flipping alg, OLL parity, PLL parity). Use for: method-level claims, terminology, how 5x5 relates to other big cubes.
- [CubeSkills — Beginner's Method for Solving the 5x5 Cube (PDF)](https://www.cubeskills.com/uploads/pdf/tutorials/beginners-method-for-solving-the-5x5-cube.pdf)
  Feliks Zemdegs' written beginner guide. Printable, algorithm-precise. Use for: cross-checking center strategy and edge-pairing moves against J Perm.
- [Sarah's Cubing Site — 5x5 Last 2 Edges](https://sarah.cubing.net/5x5x5/l2e)
  Comprehensive L2E case list. Overkill for a beginner solve but authoritative. Use for: looking up a specific last-two-edges case when stuck.

## Wisdom (Communities)

- [r/Cubers](https://reddit.com/r/Cubers)
  The main cubing subreddit; friendly to beginner big-cube questions, daily discussion thread for quick asks. Use for: "am I doing this pairing right" posts, hardware questions.
- [SpeedSolving.com forums](https://www.speedsolving.com/forums/)
  Older, deep community attached to the wiki. Use for: method questions with searchable history.

## Gaps

- No interactive 5x5 scramble/practice web tool vetted yet — worth finding one for drilling centers in isolation.
