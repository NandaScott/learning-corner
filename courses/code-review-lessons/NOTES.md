# Notes — Code Review Lessons

Source of the course: `lesson-briefs.md` at the repo root, generated by `pr-to-lessons` from the PR.

## Course shape

Seven principle-lessons in four arcs, then an interleaved quiz. Two references: a glossary and a pre-PR self-review checklist (the compressed, printable payoff).

- **Arc 1 — Adapting existing code** (the PR's core sin): L1 copy-paste fit, L2 layer that owns the logic.
- **Arc 2 — Clarity**: L3 names & docstrings track scope, L4 minimal explicit change.
- **Arc 3 — Coupling & horizon**: L5 temporal coupling (co-locate), L6 YAGNI (write for the current design).
- **Arc 4 — Proof**: L7 test new behaviour even when neighbours lack tests.

## Teaching decisions

- **No forward references.** Each principle is independent, so ordering is by theme, not dependency. Every recall widget is answerable from the current lesson alone. The L8 quiz is the only place they interleave.
- **Don't re-teach Clean Code.** The learner has the canon. Open on the real mistake, land the *judgment* and the *tell*, skip the SRP lecture. Briefs 2 and 3 map onto Clean Code the learner already knows — the lesson value is the specific misfire, not the principle.
- **Every lesson closes on an applied habit**, phrased as the check to run at self-review time (the "How to apply" line from each brief). That's the reflex the mission wants.
- **Django facts are verified, not recalled.** L4's `Model._meta.get_field().validators` claim was checked against the project's Django 6.0.6 source (`field.validators == [*default_validators, *_validators]`).

## Voice

Lesson prose follows `article-voice` (plain third-person lecture) over the `ai-tells` blocklist. The marked **key line** callout may stay punchy. Ran the de-slop proofread over each lesson's body prose.

## Attribution

Public-repo scrub: reviewer names, org/repo, PR number, commit SHA, and GitHub URLs are stripped from every course file. Catches are cited generically as "a real code review." Keep it that way. The pedagogical point (a real reviewer saw the thing) survives without naming anyone.
