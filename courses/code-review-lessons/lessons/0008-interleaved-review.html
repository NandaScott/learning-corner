Interleaved review

Re-reading the lessons builds fluency, the feeling of knowing. Retrieving each judgment cold, with the seven shuffled together, builds storage strength: the reflex that actually fires at self-review time.

How to use this

Questions are drawn from all seven lessons and shuffled every run, by design. Interleaving the principles, rather than reviewing one lesson at a time, is harder, and that difficulty is what makes them stick.1 Most questions pit the right answer against a near-miss, the shallow version that sounds right. Answer from memory before reading the feedback. Then hit Shuffle & retry, and run it again in a few days. The spacing is the second half of durable memory. These are habits, not facts, so what you are really drilling is which check to run at the moment the mistake would happen.

Go deeper. Missed the same lesson twice? Flag which one. It's worth re-explaining a different way, or drilling just that check. The real test isn't this quiz, it's the next PR: run the self-review checklist on your own diff and see which of the seven habits actually fired.
