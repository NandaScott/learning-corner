Interleaved review

Re-reading feels like learning but mostly builds fluency — the illusion of mastery. Retrieving from memory, cold, with topics shuffled, is what builds storage strength. That's the whole point of this page.

How to use this

Questions are drawn from all seven lessons and shuffled every run — by design. Interleaving (mixing topics rather than blocking them) is harder than reviewing one lesson at a time, and that difficulty is exactly what makes it stick.1 Many questions pit the right answer against a near-miss — the shallow version that sounds right. Answer from memory before reading the feedback. Then hit Shuffle & retry and run it again in a few days — that spacing is the second half of durable memory.

Go deeper. Missed the same lesson twice? Flag which one — it's worth re-explaining a different way, or drilling just that concept. When you can clear this cold two sessions apart, you can explain Celery-on-Redis from the metal up without notes.
